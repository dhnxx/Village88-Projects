CodeIgniter Mini Documentation

Controllers

Controllers in CodeIgniter are PHP classes responsible for handling the incoming HTTP requests and executing the corresponding application logic. 


            <?php
            class Welcome extends CI_Controller {
                public function index() {
                    // Controller logic here
                }
            }
            ?>

Views

Views in CodeIgniter are PHP files responsible for presenting the user interface to the client. They contain HTML along with embedded PHP code to dynamically generate content. 


            <!-- File: views/welcome_message.php -->
            <h2><?php echo $title; ?></h2>
            <p>Welcome to CodeIgniter</p>

Models

Models in CodeIgniter are PHP classes responsible for interacting with the database. They encapsulate the data access logic and provide an interface for retrieving and manipulating data. 


            <?php
            class User_model extends CI_Model {
                public function get_users() {
                    // Model logic to retrieve users from the database
                }
            }
            ?>

Sessions

Sessions in CodeIgniter provide a way to store user data across multiple HTTP requests. They allow you to maintain stateful information such as user authentication status, shopping cart items, etc. 


            <?php
            // Setting session data
            $this->session->set_userdata('user_id', 123);
            
            // Retrieving session data
            $user_id = $this->session->userdata('user_id');
            ?>

Input Class

The Input Class in CodeIgniter provides methods for retrieving user input data from various sources such as POST, GET, COOKIE, etc. 


            <?php
            // Retrieving POST data
            $username = $this->input->post('username');
            
            // Retrieving GET data
            $user_id = $this->input->get('user_id');
            ?>

Form Helper

The Form Helper in CodeIgniter simplifies working with forms. It offers functions for creating form elements:


            // Load Form Helper
            $this->load->helper('form');

            // Create form input field
            $data = array(
                'name' => 'username',
                'id' => 'username',
                'value' => 'johndoe',
                'maxlength' => '100',
                'size' => '50',
                'style' => 'width:50%'
            );
            echo form_input($data);

            // Create dropdown menu
            $options = array(
                'small' => 'Small Shirt',
                'med' => 'Medium Shirt',
                'large' => 'Large Shirt',
                'xlarge' => 'Extra Large Shirt'
            );
            $shirts_on_sale = array('small', 'large');
            echo form_dropdown('shirts', $options, 'large');

URL Helper

The URL Helper in CodeIgniter provides functions for generating URLs and managing URL-related tasks. 


            <?php echo base_url('assets/css/main.css'); ?>
